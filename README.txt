
The files contained in this repository are the datafiles for the project ''A multiomics disease progression signature of low-risk ccRCC'', as described in 
the paper itself. 

The file marked A multiomics disease progression signature of low-risk ccRCC_miR_data contains the data from the miRNA seq, while the other files contain 
the mRNA and Protein data respectively. The details for each dataset and the patient cohort are described in the paper itself.
